Tentang Saya

Saya adalah seorang edukator dan developer yang berfokus pada pembuatan solusi teknologi untuk dunia pendidikan. Dengan pengalaman lebih dari 5 tahun, saya percaya bahwa teknologi dapat menjadi jembatan untuk mengatasi kesenjangan akses pendidikan di Indonesia.

Melalui platform GuruDigital, saya mengembangkan berbagai proyek mulai dari sistem e-learning, quiz interaktif, hingga dashboard analitik yang membantu guru dan siswa belajar lebih efektif.

Panduan Markdown

Referensi cepat sintaks Markdown yang didukung di kolom KontenMarkdown Google Sheet Anda.

Elemen Sintaks Markdown Keterangan
Heading # Judul Gunakan 1-6 tanda # untuk H1-H6
Bold **teks tebal** Menghasilkan teks tebal
Italic *teks miring* Menghasilkan teks miring
Link [teks](url) Contoh: [Google](https://google.com)
Image ![alt](url) Tampilkan gambar dari URL
List - item Gunakan - atau 1. untuk daftar
Code `kode` Inline code dengan backtick
